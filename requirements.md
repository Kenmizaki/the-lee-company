# The Lee Company Japan - 日本語サイト要件定義書

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| サイト名 | The Lee Company Japan |
| ドメイン | theleecompany.co.jp |
| 目的 | 日本市場での新規商談獲得 |
| ターゲット | 流体制御部品を必要とする日本企業のエンジニア・購買担当者 |
| 言語 | 日本語のみ |
| 運用主体 | 株式会社ジュピターコーポレーション（日本総代理店） |
| 最終形態 | WordPress（静的HTMLで構築後、移行） |
| 参照元 | https://www.theleeco.com/ |

---

## 2. デザイン方針

- 本国サイト（theleeco.com）のデザインを踏襲
- ロゴ・画像素材は本国サイトのものを使用
- 新規デザインは作成しない
- 情報量は本国より削減してシンプルに

## 14. テンプレート設計（WordPress移行を見据えて）

### 14.1 製品ページテンプレート

WordPress移行後、ノーコードで追記・修正できる構造にする。

**固定要素（テンプレート側）**
- ヘッダー
- パンくずリスト
- ページタイトル（H1）
- CTA
- フッター

**可変要素（投稿コンテンツ側）**
- 製品概要テキスト
- 製品画像
- 特徴・仕様
- 用途・適用分野

### 14.2 産業ページテンプレート

**固定要素（テンプレート側）**
- ヘッダー
- パンくずリスト

- 関連製品セクション（※任意）
- CTA
- フッター

**可変要素（投稿コンテンツ側）**
- 産業概要テキスト
- 産業イメージ画像
- 対応分野・用途
